## 🧠 Ce que c'est

Plateforme domotique open source qui centralise tous les appareils connectés du domicile dans un seul dashboard local : lumières, prises, thermostats, caméras, alarmes, volets, robots aspirateurs... 3000+ intégrations. Fonctionne entièrement en local, sans cloud, sans abonnement.

---

## ⚙️ Fonctionnalités principales

- **3000+ intégrations** : Philips Hue, IKEA, Sonos, Netatmo, Shelly, Tuya, Zigbee, Z-Wave, Matter, Nest...
- **Automatisations** : règles visuelles (si mouvement ET heure < 22h → allumer)
- **Dashboard Lovelace** : entièrement personnalisable
- **Energy monitoring** : consommation électrique temps réel
- **Voix locale** : Whisper (reconnaissance) + Piper (synthèse)
- **Add-ons** : Frigate (vision IA caméras), ESPHome, Node-RED, HACS
- **Apps iOS/Android** : géolocalisation + contrôle distant

---

## 🚀 Installation

**Recommandé : Home Assistant OS sur Raspberry Pi 4/5 ou mini-PC**
```
Télécharger l'image : https://www.home-assistant.io/installation/raspberrypi
Flasher sur carte SD avec Raspberry Pi Imager
Dashboard : http://homeassistant.local:8123
```

---

## 📌 Pertinence

Référence absolue de la domotique open source. Remplace SmartThings, Apple Home (partiellement), Alexa hub — sans abonnement et sans données qui quittent la maison. Hardware : Raspberry Pi 4/5 (~80€) ou mini-PC.
