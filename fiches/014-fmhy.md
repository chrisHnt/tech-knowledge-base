## Ce que c'est

FMHY (freemediaheckyeah) est le plus grand annuaire communautaire de ressources gratuites sur internet. Il référence des milliers de liens organisés par thématiques : streaming, téléchargement, gaming, lecture, IA, privacy, outils tech, éducation, etc. Le projet est maintenu sur GitHub et publié sur fmhy.net, avec une communauté active sur Reddit et Discord.

## Fonctionnement

- **Wiki/annuaire** : chaque page thématique liste des services, outils et sites avec indicateurs de qualité (⭐ = recommandé)
- **SafeGuard** : extension navigateur compagnon qui avertit si un site listé est devenu dangereux
- **Startpage** : page de démarrage navigateur pré-configurée
- **SearXNG** : instance de recherche privée hébergée par la communauté (searx.fmhy.net)
- **Backups** : miroirs réguliers du contenu pour résilience
- **Contributions** : PR GitHub ouvertes + Discord actif pour proposer de nouveaux liens

## Points notables

- Section **Artificial Intelligence** (fmhy.net/ai) très complète : chatbots gratuits, générateurs d'images, outils IA, LLMs locaux, comparatifs modèles
- Section **Privacy / Adblocking** couvre les outils de protection vie privée
- Section **Developer Tools** pour outils de développement
- Section **Linux / macOS** pour l'écosystème open source
- Index mis à jour en continu par la communauté
- Version SFW disponible sur fmhy.xyz

## Usage rapide

```
https://fmhy.net/ai               → ressources IA gratuites (LLMs, chatbots, outils)
https://fmhy.net/privacy          → adblock, vie privée
https://fmhy.net/developer-tools  → outils dev
https://searx.fmhy.net            → moteur de recherche privé
https://github.com/fmhy/FMHY-SafeGuard → extension sécurité navigateur
```

## Pertinence pour ce projet

Utile comme **méta-référence** pour découvrir de nouveaux outils à ficher. La section IA (fmhy.net/ai) peut servir de radar pour identifier des LLMs gratuits, interfaces d'inférence, ou frameworks à explorer et documenter dans l'ai-knowledge-base. À consulter ponctuellement, pas comme source primaire d'architecture.
